<i>Faça um programa para ler a cotação do dólar, e depois um valor em dólares a ser comprado por uma pessoa em reais.
Informar quantos reais a pessoa vai pagar pelo dólares, considerando ainda que a pessoa teá que pagar 6% de IOF sobre o valor em dólar. Criar uma classe CurrencyConverter para ser responsável pelos cálculos.

Exemplo: <br>
What is the dollar price? 5.15 <br>
How many dollar wil be bought? 200.00 <br>
Amoun to be paid in reais = 657.20 <br> </i>

```
package entities;

public class CurrencyConverter {
	private final double dolar = 5.15;
	private static final double IOF = 0.06;
	public static int cotacoesFeitas;
	
	
	public double precoParaOReal(double dolar, double dolarComprado) {
		double basic = dolar * dolarComprado;
		return basic;
	}


	public double getDolar() {
		return dolar;
	}


	public static double getIOf() {
		return IOF;
	}
	
	public double cotacaoComIOF(double IOF) {
		return IOF;	
	}	
}
```

	package application;
	
	
	import java.util.Scanner;
	
	import entities.CurrencyConverter;

	
	
	public class Program {

			System.out.println(cotacao.precoParaOReal(cotacao.getDolar(), dolarCompra) 
					+ cotacao.precoParaOReal(cotacao.getDolar(), dolarCompra)

			sc.close();
		}
